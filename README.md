Пользовательское соглашение Nulls Messenger

Дата последнего обновления: 10 марта 2026 г.

Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует использование приложения Nulls Messenger (далее — «Сервис», «Мессенджер»). Используя Сервис, вы соглашаетесь с условиями данного Соглашения.

1. Общие положения

1.1. Nulls Messenger — это программное обеспечение для обмена сообщениями, файлами и мультимедийным контентом между пользователями.
1.2. Использование Сервиса означает полное согласие пользователя с данным Соглашением.
1.3. Если пользователь не согласен с условиями, он должен прекратить использование приложения.

2. Регистрация и аккаунт

2.1. Для использования некоторых функций может потребоваться создание аккаунта.
2.2. Пользователь обязуется предоставлять достоверную информацию при регистрации.
2.3. Пользователь несёт ответственность за сохранность данных для входа в аккаунт.

3. Правила использования

Пользователь обязуется не использовать сервис для:

распространения незаконного контента

отправки спама или вредоносного ПО

нарушения прав других пользователей

публикации материалов, нарушающих авторские права

попыток взлома, изменения или нарушения работы сервиса

Администрация вправе ограничить или заблокировать доступ пользователя при нарушении правил.

4. Контент пользователей

4.1. Пользователь сохраняет права на отправляемый контент.
4.2. Пользователь предоставляет сервису право хранить и передавать сообщения для обеспечения работы мессенджера.
4.3. Администрация не несёт ответственности за контент, отправленный пользователями.

5. Конфиденциальность

5.1. Сервис может собирать технические данные для улучшения работы приложения.
5.2. Персональные данные обрабатываются в соответствии с политикой конфиденциальности.
5.3. Администрация принимает разумные меры для защиты данных пользователей.

6. Ограничение ответственности

6.1. Сервис предоставляется «как есть».
6.2. Администрация не гарантирует бесперебойную работу сервиса.
6.3. Администрация не несёт ответственности за убытки, возникшие при использовании сервиса.

7. Изменения соглашения

7.1. Администрация вправе изменять настоящее Соглашение.
7.2. Обновлённая версия вступает в силу после публикации.

8. Прекращение использования

Пользователь может прекратить использование сервиса в любое время, удалив приложение или аккаунт.

9. Контакты

По вопросам, связанным с использованием сервиса, пользователь может обратиться к администрации через официальные каналы поддержки.
